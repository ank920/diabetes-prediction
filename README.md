  <section id="overview">
    <h2>Overview</h2>
    <p>
      This project uses a Gaussian Naive Bayes classifier to predict whether an individual has diabetes based on a set of medical features.
      It demonstrates both an interactive command-line interface and a REST API built with Flask, making it easy to test and deploy the model.
    </p>
    <a href="#top" class="back-to-top">Back to Top</a>
  </section>

  <section id="features">
    <h2>Features</h2>
    <ul>
      <li><strong>Data Preprocessing:</strong> Uses <code>StandardScaler</code> to normalize features.</li>
      <li><strong>Model Training:</strong> Splits the dataset and trains a Gaussian Naive Bayes model.</li>
      <li><strong>Interactive CLI:</strong> Prompts the user to input feature values for prediction.</li>
      <li><strong>REST API:</strong> Provides a Flask endpoint (<code>/predict</code>) for predictions.</li>
    </ul>
    <a href="#top" class="back-to-top">Back to Top</a>
  </section>

      <pre>python3 -m venv env

      <pre>pip install pandas scikit-learn flask</pre>
      <h3>Step 4: Add the Dataset</h3>
      <p>Place your <code>diabetes.csv</code> file in the project root directory.</p>

  <section id="usage">
    <h2>Usage</h2>
    <button type="button" class="collapsible">Expand Usage Instructions</button>
    <div class="content">
      <h3>Interactive Prediction</h3>
      <ol>
        <li>Ensure the <code>predict_user_input()</code> call in <code>main.py</code> is uncommented.</li>
        <li>Run the script:
          <pre>python main.py</pre>
        </li>
        <li>Follow the on-screen prompts to input feature values.</li>
      </ol>
      <h3>Running the Flask API</h3>
      <ol>
        <li>Comment out or remove the <code>predict_user_input()</code> call in <code>main.py</code>.</li>
        <li>Run the Flask app:
          <pre>python main.py</pre>
        </li>
        <li>Access the <code>/predict</code> endpoint with a POST request containing the JSON payload.</li>
      </ol>
    </div>
    <a href="#top" class="back-to-top">Back to Top</a>
  </section>

  <section id="testing">
    <h2>Testing the API</h2>
    <p>You can test the <code>/predict</code> endpoint using curl or Postman.</p>
    <h3>Using curl:</h3>
    <pre>
curl -X POST "http://127.0.0.1:5000/predict" \
     -H "Content-Type: application/json" \
     -d '{"Pregnancies": 3, "Glucose": 120, "BloodPressure": 70, "SkinThickness": 30, "Insulin": 80, "BMI": 25.0, "DiabetesPedigreeFunction": 0.5, "Age": 32}'
    </pre>
    <p>The expected response is a JSON object, e.g.:
    <pre>{"prediction": "No Diabetes"}</pre>
    </p>
